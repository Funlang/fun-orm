# fun-orm

# Fun 语言 O/R Mapping 框架

## 框架功能特性

支持通过 Fun 语言以 O/R Mapping 的方式访问存取数据库.

1. 支持 ADO, OleDB, ODBC 访问数据库;
2. 支持文本数据库(*.txt, *.csv);
3. 支持文件数据库(*.dbf);
4. 支持 Access, SQLServer, MySQL, Oracle, DB2, SQLite 等数据库;
5. 支持以对象方式增删查改数据库记录;
6. 支持主键/外键查询;
7. 支持 RPC 方式访问;
8. 支持复杂查询条件(类似 MongoDB);
9. 支持事务处理.
